# Recipe Finder Web APp With Vue

Simple web App for finding food recipes made with VueJs and Bootstrap for submission Dicoding Front-end Developer

## Installation

How to Install :

```bash
npm install
```

## Usage

Start development :

```bash
npm run start-dev
```

How to build:

```bash
npm run build
```
